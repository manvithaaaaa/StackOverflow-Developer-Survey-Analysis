<h1>Stack Overflow Developer Survey Analysis</h1>

<h2>Problem Statement</h2>
The Stack Overflow Developer Survey is one of the largest developer surveys in the world with 65,000+ responses. This project analyzes the survey data to uncover trends in programming languages, tools, salaries, AI adoption 
and developer demographics.

<h2>Objective</h2>
Extract meaningful insights from the survey data to understand the current state of the developer ecosystem.


 <h2>Tools & Technologies</h2>
<ul>
  <li>Python, Pandas, NumPy</li>
<li>Matplotlib, Seaborn, Plotly</li>
<li>Jupyter Notebook</li>
</ul>
